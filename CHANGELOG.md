# Changelog

## 2026-04-13
- 主导航底部栏改为 `flutter_floating_bottom_bar`，随滚动显隐；`IndexedStack` 各 Tab 仅当前页绑定滚动控制器，避免多列表共用一个 `ScrollController`导致切换异常。
- 搜索首页：未输入时采用与历史搜索页类似的沉浸式顶栏 + 内嵌搜索条，并展示「为你推荐」分页与「浏览」分类网格；输入后仍为原有分类建议列表样式。
- 搜索首页移除独立「最近搜索」区块；本地搜索历史改为聚焦时在输入框下方以建议列表展示，点击即发起搜索。
- 新增 `SearchKeywordHintsService`：在推荐、探索接口成功拉取媒体列表后，将标题等字段去重合并为本地关键字提示（SharedPreferences 持久化，与 Realm 搜索历史合并展示），便于快速选词。

## 2026-04-08
- iOS 订阅日历 Widget 补全实现与展示逻辑，增强稳定性与可用性。
- 登录与自动登录链路新增站点数据预热，减少 Widget/站点相关模块首次进入时的等待与失败概率。
- 文档补充 HarmonyOS（OHOS）分支与自签说明。

## 2026-04-03
- 媒体整理页新增列表/按标题聚合两种显示方式，支持右上角快速切换并记住聚合视图折叠状态。
- 媒体整理页聚合视图支持分组多选、反选与批量删除，选择态下同步收敛顶部筛选、条目操作入口与底部操作栏交互。
- 媒体整理页界面细节优化：分组卡片、选择入口、右上角视图切换按钮与卡片选中态样式统一收敛。
- 媒体整理页删除完成后会自动重置筛选条件到全部，并在进入远端加载前强制刷新一次图片 Cookie。
- 媒体搜索列表页在请求搜索结果前新增图片 Cookie 强制刷新，降低海报加载失败概率。
- 下载器直连下载增强：兼容普通直链与带 enclosure 请求描述的下载链路，支持按 method、header、params 解析后再获取种子文件。
- 修复部分站点直连下载第一跳参数位置不匹配的问题，接口返回“参数错误”时会自动改用 query 参数重试。
- 下载器直连下载补充关键链路日志，便于排查 enclosure 解析、第一跳取链与第二跳下载失败原因。
- Dashboard 编辑页保存逻辑改为纯本地配置管理，移除远端 Dashboard 配置与排序同步链路。
- Dashboard 编辑页内部组件收敛为 Material 交互，统一选择项卡片点击反馈与状态表现。
- Dashboard 编辑页右上角保存态 loading 调整为更轻量的 `CupertinoActivityIndicator`。
- Dashboard 数据刷新链路优化：移除重复定时刷新，避免同批数据被高频重复拉取。
- 修复 Dashboard 页面错误重复创建控制器与 Realm 服务实例的问题，减少状态分叉与额外开销。
- 调整 Dashboard 刷新范围，仅按当前展示组件加载对应数据，减少不必要请求。

## 2026-04-01
- 下载管理页种子列表重构为更紧凑的信息卡布局，收敛标题、进度、统计项与底部元信息的展示密度。
- 下载管理页种子统计区改为自然换行布局，移除固定宽度裁切并提升统计文案字号与可读性。
- 下载管理页种子进度显示优化：0 进度任务保留最小可见进度条，文案仍显示为 0。
- 下载管理页宽屏布局适配：屏幕宽度大于 600 时，种子列表改为双列网格展示。
- 修复 qBittorrent 增量同步时热度、可用性等字段偶发丢失导致列表信息时有时无的问题。
- 调整种子列表底部做种/下载人数展示口径，改为使用总做种/总下载数而非当前连接 peers 数。
- 下载器配置列表样式重构：卡片信息层级与运行指标区重新设计，增强与站点管理页的视觉区分。
- 下载器指标区新增合并分栏展示（总下载、总上传、上/下比），并统一“上/下比”计算口径为累计上传/累计下载。
- 下载器配置列表新增隐私模式开关，支持对 URL 地址信息进行模糊遮挡显示。
- 隐私模式开关调整为 `title + switch` 交互样式，并支持本地持久化恢复。
- 媒体整理页加载链路优化：进入页面优先展示本地缓存，再请求远程最新数据，减少首屏等待体感。
- 媒体整理页新增历史缓存读写能力（按关键词维度缓存），远程请求失败时保留已展示数据避免列表闪空。
- 媒体整理页下拉刷新改为并行刷新存储与历史数据，缩短刷新耗时。
- 媒体整理页信息栏加载态调整为左侧 `CupertinoActivityIndicator`，移除右上角独立 loading 图标。
- 媒体整理记录项操作模式文案本地化：`move/copy/hard link/soft link` 显示为中文。
