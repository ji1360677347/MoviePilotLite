# Changelog

## 2026-06-25
- 新增 Android APK 热更新能力：应用设置页“当前版本”支持检查远程发布版本、下载 APK、查看下载进度并调起系统安装器。
- Android 更新检查改为仅识别 `release-v版本-日期` 格式的 Release，避免误将 HarmonyOS/OHOS 发布识别为安卓版本。
- 修复 Android Release tag 中日期被误识别为构建号的问题，版本比较仅使用实际版本号。
- GitHub Release 请求支持携带系统基础设置中的 Github Token，降低匿名请求触发 API 限流的概率，并补充限流场景提示。
- APK 下载支持应用内后台继续下载，进度面板可收起后再次从版本入口打开。
- APK 下载新增取消能力，取消或失败时会清理未完成安装包，避免残留损坏文件。
- iOS 设备检查更新时仅展示 Release 更新日志，并提供前往 GitHub Release 页面入口，不触发 APK 下载。
- App 启动时自动清理本地更新缓存中的历史 APK 文件，减少无效安装包占用空间。
- Android 原生侧补齐 APK 安装权限、FileProvider 与安装通道，兼容未知来源安装授权流程。
