- **Rule Module**: Implements a unified `RuleController` parameterized by `RuleType` (custom, priority, download) to manage distinct rule sets via dedicated API endpoints. It uses `freezed` for immutable data models and separates UI into specific pages (`CustomRulePage`, `DownloadRulePage`) with shared widgets like `RuleCard`.
- **Multifunction Module**: Acts as a central dashboard hub using `MultifunctionController` to aggregate real-time metrics from subscriptions, downloads, plugins, and sites. It employs periodic polling for downloader stats and constructs dynamic `DashboardModuleViewModels` for a responsive, grid-based UI in `MultifunctionPage`.
- **User Management Module**: Provides administrative capabilities via `UserManagementController`, which fetches user lists and enriches them with per-user subscription statistics. It enforces superuser access checks and supports search and deletion operations through a dedicated page and item cards.
